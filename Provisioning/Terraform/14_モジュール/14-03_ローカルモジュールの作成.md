# ローカルモジュールの作成

チーム内で S3 バケットを作るたびに「暗号化設定を忘れた」「バージョニングが入ってない」が起きているなら、正しい設定を込みにしたモジュールを作って配布する方がいい。呼び出す側はバケット名を渡すだけで済む。

---

## 最小構成

```
modules/
└── s3_with_logging/
    ├── main.tf       # リソース定義
    ├── variables.tf  # 入力変数
    └── outputs.tf    # 出力値
```

---

## variables.tf

```hcl
variable "bucket_name" {
  type        = string
  description = "S3 バケット名"
}

variable "tags" {
  type        = map(string)
  description = "共通タグ"
  default     = {}
}
```

---

## main.tf

```hcl
resource "aws_s3_bucket" "main" {
  bucket = var.bucket_name
  tags   = var.tags
}

resource "aws_s3_bucket_versioning" "main" {
  bucket = aws_s3_bucket.main.id
  versioning_configuration {
    status = "Enabled"
  }
}

resource "aws_s3_bucket_server_side_encryption_configuration" "main" {
  bucket = aws_s3_bucket.main.id
  rule {
    apply_server_side_encryption_by_default {
      sse_algorithm = "AES256"
    }
  }
}
```

---

## outputs.tf

```hcl
output "bucket_id" {
  value = aws_s3_bucket.main.id
}

output "bucket_arn" {
  value = aws_s3_bucket.main.arn
}
```

---

## 呼び出し方

```hcl
module "app_bucket" {
  source      = "./modules/s3_with_logging"
  bucket_name = "myapp-data"
  tags = {
    Env = "production"
  }
}

output "app_bucket_arn" {
  value = module.app_bucket.bucket_arn
}
```

モジュールを変更したら `terraform init` を再実行する（ローカルモジュールは init 不要の場合もあるが、確実に実行する）。
