# コマンドとショートカット早見表

## 使い方

インタラクティブセッション中に使えるコマンド・ショートカットをまとめたリファレンス。

---

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Esc` | 現在の処理をキャンセル（Claude が作業中のとき止める） |
| `Esc` → `Esc`（2回） | 直前のリクエストを取り消して再入力 |
| `Ctrl + C` | セッションを終了 |
| `Ctrl + R` | コマンド履歴を検索 |
| `Shift + Tab` | 権限モードをサイクル（default → acceptEdits → plan） |
| `\` + `Enter` | 改行（複数行入力） |

---

## 入力プレフィックス

| プレフィックス | 動作 |
|--------------|------|
| `@ファイル名` | ファイルをコンテキストに追加（補完あり） |
| `!コマンド` | シェルコマンドを直接実行（Claude のトークンを消費しない） |
| `/コマンド名` | スラッシュコマンドを実行 |

---

## セッション管理

| コマンド | 動作 |
|---------|------|
| `/clear` | 会話履歴をリセット |
| `/compact` | 会話を要約して圧縮（コンテキスト節約） |
| `/resume` | 前回のセッションを再開 |
| `/exit` | セッションを終了 |

---

## 情報確認

| コマンド | 動作 |
|---------|------|
| `/help` | コマンド一覧を表示 |
| `/status` | 現在のセッション状態を確認 |
| `/cost` | トークン使用量を確認（API 課金ユーザー向け） |
| `/stats` | 使用パターンを確認（サブスクリプション向け） |
| `/doctor` | 設定の問題を診断 |

---

## 設定・カスタマイズ

| コマンド | 動作 |
|---------|------|
| `/model` | 使用モデルを切り替え |
| `/effort low\|medium\|high\|max` | 思考の深さ（Effort Level）を変更 |
| `/fast` | Fast Mode をトグル（Opus 専用・高速高コスト） |
| `/permissions` | ツール権限の確認・変更 |
| `/config` | 設定を確認・変更 |
| `/memory` | メモリファイルを管理 |
| `/init` | CLAUDE.md を自動生成 |

---

## エージェント・自動化

| コマンド | 動作 |
|---------|------|
| `/agents` | サブエージェントを管理 |
| `/plan` | Plan Mode に入る（読み取り専用） |
| `/mcp` | MCP サーバーの接続状況を確認 |
| `/hooks` | フックを管理 |

---

## エクスポート

| コマンド | 動作 |
|---------|------|
| `/export` | 会話内容をファイルに書き出す |

---

## CLI フラグ（起動時）

```bash
claude                          # 通常起動
claude "プロンプト"              # 1 ターンだけ実行して終了
claude -p "プロンプト"           # Headless Mode
claude --model claude-opus-4-6  # モデル指定
claude --effort high            # Effort Level 指定
claude --permission-mode plan   # Plan Mode で起動
claude -c                       # 前のセッションを継続
claude -r                       # セッションを選んで再開
claude --bare                   # hooks 等をスキップ（CI 向け）
claude --add-dir /path/to/dir   # 追加ディレクトリをコンテキストに含める
```
