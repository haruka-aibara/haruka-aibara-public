# アドバンスト機能の概要

## このセクションについて

基本的な使い方（CLAUDE.md・スキル・権限・プロンプト）を押さえた後に覚えると効果的な機能群。

それぞれ独立した機能だが、組み合わせることで Claude Code を「個人のアシスタント」から「自律的なチームメンバー」レベルの使い方に引き上げられる。

---

## 各機能と使いどころ

| 機能 | 何ができるか | 使いどころ |
|------|------------|-----------|
| **ステータスラインのカスタマイズ** | ターミナルに常時表示する情報をカスタマイズ | モデル・コスト・セッション状態を一目で確認したい |
| **MCP（外部ツール連携）** | GitHub・Jira・Slack 等を Claude から直接操作 | ツール間の往復を減らしたい |
| **スケジュール実行** | 定期的なタスクを自動化 | 週次レポート・依存関係チェックを人手なしで回したい |
| **モデル設定** | 用途に合わせてモデル・Effort Level を調整 | コストと品質のバランスを最適化したい |
| **出力スタイル** | Claude の回答スタイルを切り替え | 学習用・コードレビュー用など場面に応じて変えたい |
| **プラグイン** | LSP 連携・外部サービス・ワークフロー拡張 | IDE 相当の機能を CLI で使いたい |
| **スケーリング** | 複数セッションの並列実行 | 大規模タスクを効率よく分割処理したい |
| **セキュリティ** | 権限・サンドボックス・プロンプトインジェクション対策 | 本番環境や CI での安全な運用 |
