# プラグインの使い方

## こういうときに使いたい

「Claude に GitHub の PR を直接操作させたい」「Slack に通知を投げさせたい」「Python を書いたら型エラーをリアルタイムで知らせてほしい」。

MCP サーバーでも外部連携はできるが、**プラグイン**はより手軽にインストールできるパッケージ化された拡張機能。コードナビゲーション・外部サービス連携・開発フローの強化など、様々なプラグインが公式・コミュニティから提供されている。

---

## プラグインのインストール

セッション内の `/plugin` コマンドで操作する。

```bash
# 公式マーケットプレイスから一覧を確認
/plugin

# インストール（例：GitHub 連携）
/plugin install github@claude-plugins-official

# インストール後、再起動なしで反映
/reload-plugins
```

## 公式プラグインの主なカテゴリ

### 外部サービス連携

| プラグイン | できること |
|-----------|-----------|
| `github` | PR の作成・レビュー・マージ操作 |
| `gitlab` | GitLab の MR・Issue 操作 |
| `atlassian` | Jira チケット・Confluence ページ操作 |
| `linear` | Linear の Issue 管理 |
| `slack` | Slack へのメッセージ投稿 |
| `notion` | Notion ページの読み書き |
| `sentry` | Sentry のエラー確認 |
| `figma` | Figma デザインの参照 |

### 開発ワークフロー強化

| プラグイン | できること |
|-----------|-----------|
| `commit-commands` | コミットメッセージ生成・管理 |
| `pr-review-toolkit` | PR レビューの自動化支援 |

## Code Intelligence プラグイン（LSP 連携）

LSP（Language Server Protocol）と連携し、Claude に **IDE 相当のコード理解力**を追加するプラグイン群。インストールすると：

- **型エラー・構文エラーを編集後に自動検出**（`Ctrl+O` で確認）
- **定義へのジャンプ・参照検索**
- **型情報の取得・シンボル一覧・呼び出し階層**

言語ごとに LSP バイナリのインストールが別途必要。

| 言語 | プラグイン名 | 必要なバイナリ |
|------|------------|--------------|
| Python | `pyright-lsp` | `pyright-langserver` |
| TypeScript / JS | `typescript-lsp` | `typescript-language-server` |
| Go | `gopls-lsp` | `gopls` |
| Rust | `rust-analyzer-lsp` | `rust-analyzer` |
| Java | `jdtls-lsp` | `jdtls` |
| C / C++ | `clangd-lsp` | `clangd` |
| C# | `csharp-lsp` | `csharp-ls` |
| Swift | `swift-lsp` | `sourcekit-lsp` |

インストール例（Python）：

```bash
# 1. LSP バイナリを入れる
pip install pyright

# 2. プラグインをインストール
/plugin install pyright-lsp@claude-plugins-official
```

---

## プラグインを入れすぎない

プラグイン・MCP サーバー・スキル・サブエージェントを大量に有効化すると、それぞれが Claude のコンテキストウィンドウを消費する。「コンテキストの汚染（context pollution）」が起きると、推論の精度が落ちたりコストが増えたりする。

**実際に使うものだけ入れる。** 「とりあえず入れておこう」は避ける。

不要になったプラグインは：

```bash
/plugin uninstall プラグイン名
```

---

## マーケットプレイスの追加

デフォルトでは公式マーケットプレイスのみ。コミュニティマーケットプレイスを追加するには：

```bash
/plugin marketplace add anthropics/claude-code
```

GitHub リポジトリ・Git URL・ローカルパスからもインストールできる。
