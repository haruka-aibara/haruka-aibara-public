# ローカル値（locals）

モジュール内で再利用する計算値や共通値を定義する仕組み。DRY（Don't Repeat Yourself）のために使う。

---

## locals ブロック

```hcl
locals {
  env         = var.env
  app_name    = "myapp"
  name_prefix = "${local.app_name}-${local.env}"

  common_tags = {
    Project     = local.app_name
    Environment = local.env
    ManagedBy   = "Terraform"
  }
}
```

参照するには `local.名前`（`locals` ではなく `local`）。

---

## 使用例

```hcl
resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t3.micro"

  tags = merge(local.common_tags, {
    Name = "${local.name_prefix}-web"
    Role = "web"
  })
}

resource "aws_s3_bucket" "logs" {
  bucket = "${local.name_prefix}-logs"
  tags   = local.common_tags
}
```

---

## 変数との違い

| | `variable` | `locals` |
|---|---|---|
| 外部から渡せる | ✅ | ❌（モジュール内のみ） |
| 式・関数が使える | ❌（default 以外は不可） | ✅ |
| 用途 | 外部からの入力 | 内部の計算・共通値 |

タグや名前プレフィックスなど「コード内で何度も繰り返す値」は `locals` にまとめるとメンテしやすい。
