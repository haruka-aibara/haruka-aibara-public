# 型制約

変数に型を指定することで、不正な値の入力を早期に検出できる。

---

## プリミティブ型

```hcl
variable "name" {
  type = string
}

variable "count" {
  type = number
}

variable "enabled" {
  type = bool
}
```

---

## コレクション型

```hcl
# リスト（同一型の順序付きコレクション）
variable "availability_zones" {
  type    = list(string)
  default = ["ap-northeast-1a", "ap-northeast-1c"]
}

# マップ（文字列キーのコレクション）
variable "tags" {
  type = map(string)
  default = {
    Env  = "dev"
    Team = "platform"
  }
}

# セット（重複なしのコレクション）
variable "allowed_ips" {
  type = set(string)
}
```

---

## 構造体型

```hcl
# object（異なる型の属性を持てる）
variable "database" {
  type = object({
    host     = string
    port     = number
    name     = string
    readonly = bool
  })
}

# tuple（固定長、各要素が異なる型）
variable "record" {
  type = tuple([string, number, bool])
}
```

---

## any 型

```hcl
variable "config" {
  type = any
}
```

型チェックを無効化する。モジュールの入口で使うと柔軟だが、バリデーションが効かなくなるため注意。
