# KICS（Keeping Infrastructure as Code Secure）

KICS は Checkmarx が開発するオープンソースの IaC セキュリティスキャナー。Terraform・CloudFormation・Kubernetes・Ansible など幅広いフォーマットに対応している。

---

## インストール

```bash
# macOS
brew install kics

# Docker
docker pull checkmarx/kics
```

---

## 基本的な使い方

```bash
# カレントディレクトリをスキャン
kics scan -p .

# 特定のディレクトリ
kics scan -p ./infra --report-formats json -o ./reports

# Terraform のみ対象
kics scan -p ./infra -t Terraform
```

## GitHub Actions への組み込み

```yaml
- name: KICS Scan
  uses: checkmarx/kics-github-action@v1.7.0
  with:
    path: ./infra
    fail_on: high
    output_path: kics-results/
    output_formats: 'json,sarif'

- name: Upload SARIF
  uses: github/codeql-action/upload-sarif@v2
  with:
    sarif_file: kics-results/results.sarif
```

`fail_on: high` で HIGH 以上の問題があると CI が失敗する。

## 特定のルールを除外する

```bash
# 特定のクエリ ID を除外
kics scan -p . --exclude-queries "e38a8e0a-b88b-4902-b3fe-b0fcb17d5c10"
```

または `.kics-ignore` ファイルをリポジトリに置く。

```
# .kics-ignore
e38a8e0a-b88b-4902-b3fe-b0fcb17d5c10  # 社内ルールで許可済みの設定
```

## 他ツールとの位置づけ

KICS は対応フォーマットの幅広さが強み。Terraform 単体なら Checkov や Trivy でも十分だが、CloudFormation や Kubernetes も一緒にスキャンしたい場合に KICS が選ばれる。
