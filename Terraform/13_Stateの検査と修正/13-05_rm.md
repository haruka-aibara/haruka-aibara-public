# terraform state rm

State からリソースを削除するコマンド。**実際のインフラは削除されない**。State の管理から外すだけ。

---

## 基本的な使い方

```bash
terraform state rm aws_instance.web
```

---

## 使いどころ

**Terraform 管理から外したい場合**

手動管理に切り戻したいリソースを State から外す。

```bash
terraform state rm aws_security_group.legacy
```

実行後、次回の `terraform plan` では削除対象として表示されなくなる（.tf に定義が残っていれば新規作成対象になる点に注意）。

**インポートのやり直し**

`terraform import` を誤ったリソースに実行した場合に State から取り除く。

---

## 注意点

- State から消えるだけで実際のリソースは残る
- `.tf` に定義が残っていると次回 apply で「新規作成」と判定される
- 操作前にバックアップを取ること

```bash
# バックアップ
terraform state pull > backup-$(date +%Y%m%d).tfstate

# rm 実行
terraform state rm aws_instance.web
```

---

## 複数リソースをまとめて削除

```bash
terraform state rm aws_instance.web aws_instance.api
```
