# Terraform のはじめ方

インストール後、最初の `.tf` ファイルを書いて `apply` するまでの流れ。

---

## 最小構成

```
myproject/
├── main.tf
└── terraform.tf   # providers と required_version
```

---

## terraform.tf（プロバイダー設定）

```hcl
terraform {
  required_version = ">= 1.5"
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 5.0"
    }
  }
}

provider "aws" {
  region = "ap-northeast-1"
}
```

---

## main.tf（リソース定義）

```hcl
resource "aws_s3_bucket" "hello" {
  bucket = "my-hello-terraform-bucket-12345"

  tags = {
    Name      = "hello-terraform"
    ManagedBy = "Terraform"
  }
}
```

---

## 実行手順

```bash
# 1. 初期化（プロバイダーのダウンロード）
terraform init

# 2. フォーマット確認
terraform fmt

# 3. 構文チェック
terraform validate

# 4. 差分確認
terraform plan

# 5. 適用
terraform apply
# "yes" を入力して実行
```

---

## 後片付け

```bash
terraform destroy
```

---

## 次のステップ

- `variables.tf` に変数を切り出す
- `outputs.tf` に出力値を定義する
- State をリモートバックエンド（S3 等）に移す
- モジュール化して再利用性を高める
