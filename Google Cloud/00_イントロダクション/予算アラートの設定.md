# 予算アラートの設定

予算アラートを設定することで、GCPの利用費用が一定の金額に達した際に通知を受け取ることができます。

## 設定手順

1. [Google Cloud コンソール](https://console.cloud.google.com/) にアクセス
2. 「**お支払い**」→「**予算とアラート**」を選択
3. 「**予算を作成**」をクリック
4. 以下の情報を設定：
   - **予算名**: 任意の名前（例: "月間予算"）
   - **予算額**: アラートを出す金額（例: 10,000円）
   - **期間**: 月間/四半期/年間から選択
   - **アラートの閾値**: 50%、90%、100%など複数設定可能
   - **通知先**: メールアドレスを指定

## 通知の設定

- **メール**: 通知を受け取るメールアドレスを追加
- **Pub/Sub**: プログラムで処理する場合に使用
- **スプレッドシート**: 費用を記録する場合に使用

## ベストプラクティス

- **段階的なアラート**: 50%、90%、100%など複数の閾値を設定
- **早期通知**: 50%や75%で通知を受け取り、予算を確認する習慣をつける
- **複数の通知先**: チームメンバーや管理者にも通知を送る

## 注意事項

- 予算アラートは**通知のみ**で、実際の利用を制限するものではありません
- アラートを受け取ったら、コンソールで費用の内訳を確認しましょう
