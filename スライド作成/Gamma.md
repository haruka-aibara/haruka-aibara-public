# Gamma.app でスライドを作る

「スライドのデザインに時間をかけたくない」「とりあえずきれいな資料のたたき台が欲しい」——Gamma はテキストを入力するだけでデザイン済みのスライドを数十秒で生成する。日本語対応、無料から使える。

---

## 料金

| プラン | 料金 | 特徴 |
|---|---|---|
| Free | 無料（400 クレジット付き） | Gamma ロゴあり、基本機能 |
| Plus | $8/月（年払い） | 無制限 AI 生成、ロゴ非表示 |
| Pro | $15/月（年払い） | 高度な AI モデル、詳細分析、API アクセス |

1 スライド生成 = 約 10 クレジット消費。Free の 400 クレジットは 3〜4 回の生成で使い切るイメージ。定期的に使うなら Pro が現実的。

---

## 基本的な使い方

### 1. 新規作成

[gamma.app](https://gamma.app) にアクセスして「New」→「Presentation」を選ぶ。

### 2. 生成方法を選ぶ

| 方法 | 向いている場面 |
|---|---|
| **Generate** | テキストプロンプトから一から生成 |
| **Import** | Word/PDF/URL を読み込んで変換 |
| **Template** | 既存テンプレートをベースにカスタマイズ |

### 3. プロンプトを入力

```
例:
「AWS のセキュリティベストプラクティスについて、
 インフラエンジニア向けに 8 枚のスライドを作成してください。
 対象: 社内勉強会
 含める内容: IAM・KMS・GuardDuty・Config」
```

枚数と言語（日本語）を設定して Generate。

### 4. 生成後の編集

各カード（スライド）は個別に編集できる。テキストを直接書き換えるか、カードの右側にある AI ボタンで「もっと短くして」「箇条書きにして」などを指示できる。

## Gamma Agent（2025年9月〜）

Gamma 3.0 で追加された機能。プレゼン全体に対して自然言語で指示を出せる。

```
「全スライドのトーンをもっと技術的にして」
「スライド 3 のグラフをもっとシンプルにして」
「競合分析のスライドを追加して、Web を調べて内容を埋めて」
```

特に「Web を調べて内容を埋めて」ができるのが強力で、情報収集 + スライド作成を一括で頼める。

## エクスポート

| フォーマット | 用途 |
|---|---|
| PDF | 印刷・メール添付 |
| PPTX | PowerPoint で編集継続 |
| PNG | 各スライドを画像として書き出し |
| Web 公開 | Gamma のリンクで共有（インタラクティブ） |

Free プランでも PDF/PPTX のエクスポートは可能（Gamma のロゴが入る）。

## 使い方のコツ

**プロンプトは具体的に**

```
✗ 「AWS について」
✓ 「AWS Well-Architected Framework の 6 本柱を、
    CTO 向けに説明する 6 枚のスライド。
    各スライドに要点 3 つと図解のプレースホルダーを入れる」
```

**Import 機能でドキュメントを変換**

既存の Word や PDF、Notion のページ URL を渡すと Gamma がスライドに変換してくれる。「資料はあるけどスライドにする時間がない」という場面に使える。

**PPTX でエクスポートして仕上げる**

Gamma でたたき台を作って PPTX に落とし、細かいデザイン調整は PowerPoint でやるのが現実的なフロー。フォント・配色の細かい制御は Gamma より PowerPoint のほうが自由度が高い。
